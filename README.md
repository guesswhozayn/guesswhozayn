### 👨‍💻 About Me

A Software Engineer based in Pakistan, focused on building high-performance web applications, decentralized blockchain systems, and intelligent automation pipelines. I specialize in bridging clean backend engineering with modern frontend interfaces, ensuring both system reliability and an exceptional user experience.

* **🔭 Current Focus:** Building **[Picket](https://picket-hr.vercel.app)** — an AI-native candidate assessment engine with asynchronous multi-agent profile auditing.
* **🌐 Web3 & Blockchain:** Built **[Attestify](https://attestify-alpha.vercel.app)** — utilizing Ethereum Soulbound Tokens (SBTs) and IPFS for permanent academic credential verification.
* **🎓 Education:** B.Sc. in Computer Science from the **University of Chakwal** (2022–2026, CGPA 3.43/4.00).
* **📄 Academic Research:** Published author in *Spectrum of Engineering Sciences* ([DOI: 10.5281/zenodo.20655578](https://doi.org/10.5281/zenodo.20655578)).
